# Smart Wallet - Supabase Auth UI

A simple single-page Sign up / Login interface wired to Supabase authentication.

## Setup

1. Open `app.js`.
2. Replace:
   - `SUPABASE_URL`
   - `SUPABASE_ANON_KEY`
3. Serve the project with any static server.

Example:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
